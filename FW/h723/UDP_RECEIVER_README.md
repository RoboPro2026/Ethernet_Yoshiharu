# STM32H723 Ethernet UDP パケット受信ツール

STM32H723から送信されるUDPパケットを受信して表示するPythonスクリプトです。

## ファイル構成

### 1. `udp_receiver.py` - 基本版
シンプルなUDPパケット受信プログラム。受信したパケットを1つずつ表示します。

**機能:**
- パケット受信時刻の表示
- カウンター値の表示（uint32_t）
- パケットロス検出
- 送信元IPアドレス/ポート表示

### 2. `udp_receiver_stats.py` - 統計情報版
統計情報を表示する高機能版。ネットワークパフォーマンスを分析できます。

**機能:**
- リアルタイム統計表示（1秒ごと更新）
- パケット/秒（pps）の計算
- 帯域幅（bps, Kbps, Mbps）の計算
- パケット損失率の計算
- ジッター（パケット間隔のばらつき）の計算
- 最終統計サマリー表示

## 使用方法

### 1. PC側の設定

#### IPアドレスの設定
PCのIPアドレスを **192.168.1.10** に設定してください。

**Mac/Linuxの場合:**
```bash
# ネットワークインターフェース名を確認
ifconfig

# IPアドレスを設定（例: en0）
sudo ifconfig en0 192.168.1.10 netmask 255.255.255.0
```

**Windowsの場合:**
1. コントロールパネル → ネットワークと共有センター
2. アダプターの設定の変更
3. Ethernetアダプタを右クリック → プロパティ
4. IPv4のプロパティで以下を設定:
   - IPアドレス: 192.168.1.10
   - サブネットマスク: 255.255.255.0

#### ファイアウォール設定
ポート **8080/UDP** を開放してください。

**Mac:**
```bash
# システム環境設定 → セキュリティとプライバシー → ファイアウォール
# → ファイアウォールオプション → Pythonを許可
```

**Linux:**
```bash
sudo ufw allow 8080/udp
```

**Windows:**
1. Windows Defender ファイアウォール
2. 詳細設定 → 受信の規則 → 新しい規則
3. ポート → UDP → 8080

### 2. STM32側の設定

STM32のIPアドレスを確認してください：
- デフォルト: 192.168.1.10（main.cで設定）
- 送信先ポート: 8080

## トラブルシューティング

### エラー: "Address already in use"
**原因:** ポート8080が既に使用されています。

**解決方法:**
```bash
# 使用中のプロセスを確認
lsof -i :8080

# プロセスを終了
kill -9 <PID>
```

### エラー: "Cannot assign requested address"
**原因:** PCのIPアドレスが192.168.1.10に設定されていません。

**解決方法:**
- IPアドレス設定を確認してください

### パケットが受信できない
**確認事項:**
1. STM32がネットワークに接続されているか
2. LANケーブルが正しく接続されているか
3. STM32とPCが同じネットワークセグメントにあるか（192.168.1.x）
4. ファイアウォールが8080/UDPを許可しているか
5. STM32が正常に起動しているか（LEDの点滅を確認）

### パケットロスが発生する
**原因:**
- ネットワーク負荷が高い
- PCの処理が追いついていない
- ケーブル/コネクタの接触不良

**解決方法:**
- LANケーブルを確認
- 他のネットワーク通信を停止
- PCの負荷を下げる

## STM32の送信設定

現在のSTM32の設定（`main.c`）:
```c
const unsigned short src_port = 12345;  // 送信元ポート
const unsigned short dst_port = 8080;   // 送信先ポート
IP4_ADDR(&dst_addr, 192, 168, 1, 10);   // 送信先IPアドレス

// uint32_tカウンター値（4バイト）を送信
// 100ms間隔で送信
```

### カスタマイズ方法

スクリプト内の以下の定数を変更できます：

```python
LISTEN_IP = "192.168.1.10"     # 受信IPアドレス
LISTEN_PORT = 8080              # 受信ポート
STATS_INTERVAL = 1.0            # 統計表示間隔（秒）
```

## 技術仕様

- プロトコル: UDP/IPv4
- デフォルト受信アドレス: 192.168.1.10:8080
- STM32送信元ポート: 12345
- パケットサイズ: 4 bytes (uint32_t カウンター)
- 送信間隔: 約100ms (10パケット/秒)
- エンディアン: リトルエンディアン
