# STM32H723 Ethernet パケットスニファー

ネットワークインターフェース上のすべてのパケットをキャプチャして表示するツールです。

## ファイル構成

### 1. `packet_sniffer.py` - Scapy版（推奨）
Scapyライブラリを使用した高機能版。**macOSで推奨**。

**特徴:**
- すべてのプロトコルをサポート
- パケット解析が詳細
- クロスプラットフォーム（Mac/Linux/Windows）

**必要なライブラリ:**
```bash
pip3 install scapy
```

### 2. `packet_sniffer_raw.py` - Raw Socket版
Raw socketを直接使用する軽量版。**Linuxで推奨**。

**特徴:**
- 外部ライブラリ不要
- 軽量・高速
- Linuxで動作（macOSでは制限あり）

## 使用方法

### 1. インターフェース名の確認

まず、使用するネットワークインターフェース名を確認します。

**Mac/Linux:**
```bash
ifconfig
```

**出力例:**
```
en5: flags=8863<UP,BROADCAST,SMART,RUNNING,SIMPLEX,MULTICAST> mtu 1500
    ether a4:83:e7:xx:xx:xx
    inet 192.168.1.10 netmask 0xffffff00 broadcast 192.168.1.255
```

→ この場合、インターフェース名は `en5`

### 2. Scapy版の使用（推奨）

#### インストール
```bash
pip3 install scapy
```

#### 実行
```bash
cd /Users/hamuchan214/Documents/GitHub/Kaniharu/FW/h723

# 管理者権限で実行（必須）
sudo python3 packet_sniffer.py

# インターフェース名を指定する場合
sudo python3 packet_sniffer.py en5
```

#### スクリプト内の設定変更

`packet_sniffer.py`を編集：

```python
# 設定
INTERFACE = "en5"              # ← 使用するインターフェース名
STM32_IP = "192.168.1.100"     # ← STM32のIPアドレス（フィルタ用）
SHOW_ALL = False               # ← True: 全パケット, False: STM32関連のみ
```

**フィルタモード:**
- `SHOW_ALL = False` + `STM32_IP`指定: STM32関連のパケットのみ表示
- `SHOW_ALL = True`: すべてのパケットを表示（大量になる可能性あり）

## トラブルシューティング

### エラー: "Permission denied"
**原因:** パケットキャプチャには管理者権限が必要です。

**解決方法:**
```bash
sudo python3 packet_sniffer.py
```

### エラー: "scapy: No module named 'scapy'"
**原因:** Scapyがインストールされていません。

**解決方法:**
```bash
pip3 install scapy
```

### エラー: "No such device"
**原因:** 指定したインターフェース名が間違っています。

**解決方法:**
1. `ifconfig`でインターフェース名を確認
2. スクリプトの`INTERFACE`変数を修正

### パケットが表示されない
**確認事項:**
1. STM32が起動しているか
2. LANケーブルが接続されているか
3. インターフェース名が正しいか
4. `SHOW_ALL = True`にして全パケット表示してみる
5. `STM32_IP`が正しいか

### macOSでRaw Socket版が動作しない
**原因:** macOSではRaw Socketに制限があります。

**解決方法:** Scapy版を使用してください。
```bash
pip3 install scapy
sudo python3 packet_sniffer.py
```

## 高度な使用例

### 特定のポートのみ表示（スクリプト修正）

`packet_handler`関数内で追加フィルタリング：

```python
def packet_handler(packet):
    # ... 既存のコード ...
    
    # ポート8080のUDPパケットのみ表示
    if UDP in packet:
        if packet[UDP].dport != 8080 and packet[UDP].sport != 8080:
            return
    
    # ... 既存の表示処理 ...
```

### パケットをファイルに保存

```bash
sudo python3 packet_sniffer.py | tee capture_$(date +%Y%m%d_%H%M%S).log
```

### リアルタイムで特定文字列を検索

```bash
sudo python3 packet_sniffer.py | grep "192.168.1.100"
```

## パフォーマンス

- **Scapy版**: 約10,000 pps まで対応
- **Raw Socket版**: 約50,000 pps まで対応

大量のパケットをキャプチャする場合は、フィルタリングを有効にしてください。
